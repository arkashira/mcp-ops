# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute Costs**: $5/user/month
- **Storage Costs**: $2/user/month
- **Bandwidth Costs**: $1/user/month

**Total Cost per Active User**:  
**$5 + $2 + $1 = $8/user/month**

### Pricing Tiers
| Tier        | Price ($/mo) | Features                                                                 |
|-------------|---------------|--------------------------------------------------------------------------|
| Basic       | $15           | Access to MCP server discovery, basic evaluation tools, community support |
| Pro         | $30           | All Basic features + advanced evaluation tools, security compliance checks, email support |
| Enterprise   | $60           | All Pro features + dedicated account manager, custom governance policies, priority support |

### Customer Acquisition Cost (CAC) Range
- **Estimated CAC**: $50 - $100 per user

### Lifetime Value (LTV) Estimate
- **Average Revenue per User (ARPU)**: 
  - Basic: $15
  - Pro: $30
  - Enterprise: $60
- **Average Customer Lifespan**: 24 months
- **LTV Calculation**:
  - Basic: $15 * 24 = $360
  - Pro: $30 * 24 = $720
  - Enterprise: $60 * 24 = $1440

### Break-even Users Count
- **Fixed Costs**: $10,000/month (estimated)
- **Contribution Margin per User**:
  - Basic: $15 - $8 = $7
  - Pro: $30 - $8 = $22
  - Enterprise: $60 - $8 = $52

**Break-even Users Count Calculation**:
- Basic: $10,000 / $7 = **1,429 users**
- Pro: $10,000 / $22 = **455 users**
- Enterprise: $10,000 / $52 = **192 users**

### Path to $10K MRR
To achieve $10K MRR, we can target the following tier and user distribution:

- **Basic Tier**: 500 users x $15 = $7,500
- **Pro Tier**: 100 users x $30 = $3,000
- **Total MRR**: $7,500 + $3,000 = **$10,500**

**Summary**: 
To reach $10K MRR, we can aim for 500 Basic tier users and 100 Pro tier users, totaling 600 active users.
